# Examples.CodeGenerator.T4.Demo

## Overview

T4 text template is a mixture of text blocks and control logic that can generate a text file.

Use T4 templates to generate text files such as web pages, resource files, and program source code in any language.

There are two kinds of T4 text templates: run time and design time

## Run time T4 text templates

Run time templates are also known as 'preprocessed' templates.
You run the templates in your application to produce text strings, as part of its output.

### Usage

To see the code generated by executing the runtime template, please run the program.

Run the following command in the terminal:

```bash
dotnet run 
```

This will display a list of generators that produce templates.

Enter any number:

```console

Select Generator:
```

## Design time T4 text templates

Design time templates define part of the source code and other resources of your application. Typically,
you use several templates that read the data in a single input file or database,
and generate some of your .cs, .vb, or other source files.

Since the design time code is generated during the build process, there is no separate execution command.

```bash
dotnet build
```
