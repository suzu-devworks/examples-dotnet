# Dependency Injection

## T4 Template

### Setup

```shell
## Packages
dotnet add package Mono.TextTemplating --version 2.2.0
dotnet restore

## Tools
dotnet new tool-manifest
dotnet tool install dotnet-t4
dotnet tool restore
```
