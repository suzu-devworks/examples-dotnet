# Examples.DependencyInjection.Autofac.Tests

## Overview and Purpose

This is a project for learning Ioc using `Autofac`

## What is Autofac?

Autofac is an addictive Inversion of Control container for .NET Core, ASP.NET Core,
 .NET 4.5.1+, Universal Windows apps, and more.

- [Welcome to Autofac’s documentation!](https://autofac.readthedocs.io/en/latest/)

> Since .NET Framework 4.0

```bash
dotnet add package Autofac
```
