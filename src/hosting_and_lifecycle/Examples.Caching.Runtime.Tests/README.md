# Examples.Caching.Runtime.Tests

## Overview and Purpose

This project is designed to help you learn caching using `System.Runtime.Caching`.

> [!WARNING]
> The output caching functionality and types in the `System.Runtime.Caching` namespace were introduced in
.NET Framework 4, primarily for use with ASP.NET.
They are now considered somewhat outdated; while `Microsoft.Extensions.Caching` is the recommended alternative.

## Test Target

- [System.Runtime.Caching](https://learn.microsoft.com/ja-jp/dotnet/api/system.runtime.caching)
