# Examples.Metaprogramming.ExpressionTrees.Tests

## Overview and Purpose

This project includes code to experiment with and learn how to implement metaprogramming
using `System.Linq.Expressions` with an Xunit runner.

## Test Target

- [System.Linq.Expressions namespace](https://learn.microsoft.com/ja-jp/dotnet/api/system.linq.expressions)

Contains classes, interfaces and enumerations that enable language-level code expressions
to be represented as objects in the form of expression trees.
