# Examples.Metaprogramming.ReflectionEmit.Tests

## Overview and Purpose

This project includes code to experiment with and learn how to implement metaprogramming
using `System.Reflection.Emit` with an Xunit runner.

## Test Target

- [System.Reflection.Emit namespace](https://learn.microsoft.com/ja-jp/dotnet/api/system.reflection.emit)

Contains classes that can output metadata and Microsoft Intermediate Language (MSIL)
and optionally generate PE files on disk
