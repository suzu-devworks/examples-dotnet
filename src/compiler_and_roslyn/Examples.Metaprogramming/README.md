# Examples.Metaprogramming

## Overview and Purpose

This project includes sample implementations such as metaprogramming helper classes in .NET.

Currently, it only contains utilities that are shared and used in each test.

## Features

### Runtime Loader

- [Runtime Loader](./Runtime/Loader/)

  > This folder contains examples of runtime code generation and dynamic assembly loading.

### Helpers

- [Helpers](./Helpers/)

  > This folder contains helper classes and utilities that can be used in various metaprogramming scenarios.
